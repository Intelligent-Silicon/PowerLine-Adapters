# Qualcomm Atheros Open Powerline Toolkit

Qualcomm supply an open source tool called Open PLC Utils found online at https://github.com/qca/open-plc-utils.

It can be compiled to run on Linux and Windows.

The Open PLC Utils can be referred to as "Qualcomm Atheros Open Powerline Toolkit" or simply "Powerline Toolkit", this repo will refer to it as the Toolkit.

The Toolkit documentation may also refer to programs or other files detailing information which is not included. This is because some of the information is commercially sensitive, only available to partners under Non-Disclosure Agreements (NDA's) and Qualcomm's own engineers will use the full version of the Toolkit for their duties like providing support to manufacturers.
