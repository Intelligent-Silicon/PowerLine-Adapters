# Download Visual Studio

### Download instructions

At the time of writing, the Visual Studio Community 2026 webpage is at https://visualstudio.microsoft.com/vs/community/ .

Click on the Download button and this takes you to https://visualstudio.microsoft.com/thank-you-downloading-visual-studio/?sku=Community&channel=Stable&version=VS18&source=VSLandingPage&passive=false&cid=2500

The Visual Studio Installer program should download straight away.


### Installation

In your Downloads folder, load the ```VisualStudioSetup.exe``` program and follow the prompts.

Once the visual Studio installer has downloaded and installed properly, you will be presented with a number of options Workloads, Individual components, Language Packs & Installation locations.

Select the Workloads option and you should then see the following groups Web & Cloud, Desktop & Mobile, Gaming and Other Toolset options.

Tick the check box in the top right of the .NET desktop development and Desktop development with C++ in the Desktop & Mobile group.

In the bottom right of the window are two buttons. The default option for the first button is ```Install while downloading```, the second button says ```Install```.

Change the first button option so it shows ```Download all, then install```.

Now click the ```Install``` button. 

This might take anything from a number of seconds to several minutes to download, it depends on how fast your internet connection speed is.

Once its finished downloading, Visual Studio will install automatically.

Once completed click the button on the far right titled Launch.

Choose the option titled Skip and add accounts later.

Then click Start Visual Studio

Finally click the button Continue with Code.

Visual Studio will appear. Right mouse click the taskbar icon and choose Pin to Taskbar.

You can now close Visual Studio and the visual Studio Installer by clicking on the Red Close window button in the top right of the windows.
